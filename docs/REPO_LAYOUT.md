# Repository Layout

## Private Repo

Path: `/home/xler/Projects/mochi_gain_hunter_private`

Purpose:

- Rust application source
- secrets-free local configuration templates

## Why The Split Exists

If a repository is public, every tracked file in that repository is public.
Keeping the code private therefore requires a separate private repository.
